# TOON Language Support for Kate

**Prerequisites**: Kate 21.08+ (or KDevelop 5.7+), `toon-lsp` in `$PATH` (`cargo install toon-lsp`).

**Setup**: Settings → Configure Kate → LSP Client → User Server Settings, add the JSON config from `toon.json`. Optionally copy `toon.xml` for syntax highlighting.

See [docs/ide-support.md](../docs/ide-support.md) for all features and usage.
