# TOON Language Support for Helix

**Prerequisites**: Helix 23.05+, `toon-lsp` in `$PATH` (`cargo install toon-lsp`).

**Setup**: Copy the `[[language]]` and `[language-server.toon-lsp]` blocks from `languages.toml` into `~/.config/helix/languages.toml`. Optionally copy `runtime/queries/toon/highlights.scm`.

See [docs/ide-support.md](../docs/ide-support.md) for all features and usage.
