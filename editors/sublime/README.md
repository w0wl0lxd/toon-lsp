# TOON Language Support for Sublime Text

**Prerequisites**: Sublime Text 4+, [LSP package](https://packagecontrol.io/packages/LSP), `toon-lsp` in `$PATH` (`cargo install toon-lsp`).

**Setup**: Install LSP via Package Control, add the client config from `LSP-toon.sublime-settings`. Copy `TOON.sublime-syntax` to `Packages/User/`.

See [docs/ide-support.md](../docs/ide-support.md) for all features and usage.
