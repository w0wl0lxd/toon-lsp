# TOON Language Support for JetBrains IDEs

**Supports**: IntelliJ IDEA, WebStorm, PyCharm, PhpStorm, RubyMine, CLion, GoLand, Rider, DataGrip.

**Install**: Search "TOON Language" in Plugins Marketplace, or install JAR from [Releases](https://github.com/toon-format/toon-lsp/releases). The LSP binary is bundled.

See [docs/ide-support.md](../docs/ide-support.md) for all features and usage.
