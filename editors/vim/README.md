# TOON Language Support for Vim

**Prerequisites**: Vim 8.0+ (or Neovim), `toon-lsp` in `$PATH` (`cargo install toon-lsp`), one of vim-lsp / coc.nvim / Asynchronous LSP Engine.

**Setup**: See `vim-lsp-toon.vim` (vim-lsp), `coc-settings.json` (coc.nvim), or `ftdetect/toon.vim` for filetype detection.

See [docs/ide-support.md](../docs/ide-support.md) for all features and usage.
